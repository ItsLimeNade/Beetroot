# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0 | :x:                |


## Reporting a Vulnerability

To report an vulnerability, you can open an issue with the `vulnerability` tag so it gets dealt with faster. 

You can also send a mail at: hello@limenade.dev

## Expected Response Times
You can expect an answer in 1-2 days after reporting the vulnerability.

Sending a mail is the fastest way to get an answer.
