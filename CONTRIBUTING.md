# Contributing to Beetroot

Thanks for your interest in contributing! Beetroot is a Discord bot for monitoring Nightscout blood glucose data.

## Before You Start

Please read our [Code of Conduct](CODE_OF_CONDUCT.md). All contributors are expected to follow it.

## Ways to Contribute

- **Bug reports** -- open an issue describing what happened, what you expected, and how to reproduce it
- **Feature requests** -- open an issue with your idea and the use case behind it
- **Code changes** -- fix a bug or build a feature (see below)
- **Documentation** -- improvements to the README or other docs are always welcome

## Submitting a Pull Request

1. Fork the repository
2. Make your changes
3. Test that the bot still works as expected
4. Open a pull request with a clear title and description of what you changed and why

Keep PRs focused on a single change. If you want to fix multiple things, open separate PRs.

## Reporting a Bug

A good bug report includes:

- What you did
- What you expected to happen
- What actually happened
- Any relevant error messages or logs

## Questions

If you have questions, you can reach out at hello@limenade.dev or open a GitHub issue.
